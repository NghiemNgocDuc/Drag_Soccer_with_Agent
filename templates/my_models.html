Board layout

board[0]     = Quan A (large pit — cannot be sown from)
board[1–5]   = Player A's 5 small pits
board[6–10]  = Player B's 5 small pits
board[11]    = Quan B (large pit — cannot be sown from)

# Each slot is a list of strings, e.g. ['dan', 'dan', 'dan']
# len(board[i]) = number of seeds in pit i

Allowed imports

random, math, copy are available. OS / network / file-system modules are blocked for safety.

Example — random valid move

import random

MODEL_NAME  = "Random"
DESCRIPTION = "Picks a random legal move"

def get_ai_move(board, score_a, score_b, is_player_a):
    pits = range(1, 6) if is_player_a else range(6, 11)
    valid = [(p, s) for p in pits for s in (1, -1) if board[p]]
    if not valid:
        return None, 1
    return random.choice(valid)
