📋 How to write your Soccer AI — code contract ▾

Your file must define exactly one function: get_ai_move(state, is_player_a). It is called once per turn and must return a 3-tuple (player_idx, angle_degrees, power). Any code outside the function (constants, helpers) is fine.

Required signature
def get_ai_move(state, is_player_a):
    # ... your strategy ...
    return player_idx, angle_degrees, power

angle_degrees → float  direction of kick

power         → float  0 – 100  (kick strength)
